# OpenScreen Chinese v1.0.5 更新日志

## 🐛 Bug 修复

### 音频设置面板显示问题修复

**问题描述：**
在录制首页选择音频选项时，音频设置面板向下展开，当麦克风设备列表较长时会遮挡下方的其他控制按钮（如录制按钮、暂停按钮等），导致用户无法看到或点击这些按钮。

**修复内容：**
1. **改变面板展开方向**：将音频设置面板从向下展开（`top-full`）改为向上展开（`bottom-full`），避免遮挡下方按钮
2. **添加最大高度限制**：设置面板最大高度为 400px，防止面板过长
3. **添加滚动功能**：当设备列表超过最大高度时，面板内容可以滚动查看
4. **优化宽度**：设置最大宽度为 280px，确保面板不会过宽

**技术细节：**
- 修改文件：`src/components/launch/AudioSettingsPanel.tsx`
- 定位方式：从 `top-full mt-3` 改为 `bottom-full mb-3`
- 添加滚动容器：`overflow-y-auto max-h-[400px]`
- 样式优化：添加 `scrollbarWidth: 'thin'` 使滚动条更美观

**用户体验改进：**
- ✅ 音频设置面板现在向上展开，不会遮挡录制控制按钮
- ✅ 即使有多个麦克风设备，也能通过滚动查看所有选项
- ✅ 面板大小适中，不会占用过多屏幕空间
- ✅ 保持了原有的视觉风格和交互逻辑
